This package contains default implementations of {@link Successor}, and {@link java.util.Comparator} for a variety of simple data types: {@link String}, {@link Integer}, {@link Long}, etc. These classes should all be final and implement the {@link java.io.Serializable} interface. Unless instances of the class have state they should also implement the {@link org.CognitiveWeb.extser.Stateless} interface, otherwise the class should implement a custom {@link org.CognitiveWeb.extser.StreamSerializer}, which should itself be {@link org.CognitiveWeb.extser.Stateless}.

{@link Coercer}, {@link Successor}, and {@link java.util.Comparator} classes should implement equals. If the implementation class is final and does not have any persistent state, then that implementation can look like

    public boolean equals( Object o ) {
        return this == o || o instanceof <ComparatorClass>;
    }
